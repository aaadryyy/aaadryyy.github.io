Comment mettre à jour les dépendances de mon app React proprement ?

  1. Depuis votre terminal, lancez pnpm update -i -L
  2. Avec Espace sélectionnez les dépendances ayant des mises à jours mineures et/ou de patchs.

    Ces changements sont sûrs et ne casseront rien puisque vous utilisez des dépendances faites par des developpeurs respectant les bonnes pratiques (n'est-ce pas ? :).

  3. Pour chaque mise à jour majeure, vérifiez le journal des modifications (= change logs) pour voir si quoique ce soit doit être mis à jour dans le code.

    Vous pouvez trouver de tels journaux en regardant les sorties dans le project Github de la dépendance, ou en cherchant littéralement "[Nom de la dépendance] [version majeure] release note".

  4. Une fois sûr que rien ne va casser, sélectionnez la dépendance (avec Espace)
  5. Une fois terminé, appuyez sur Entrée pour installer toutes les dépendances
  6. Puis testez l'app en local pour être sûr que rien ne soit cassé
  7. Faites une demande de tirage (= Pull Request) pour que vos collègues jettent un coups d'oeil...

Et voilà !
