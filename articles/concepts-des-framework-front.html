La mémoïsation

C'est une technique d'optimisation qui vise à éviter de rendre à nouveau des composants qui ne doivent pas l'être.

L'idée va être de mettre en cache des valeurs retournées par une fonction, en fonction de ses valeurs d'entrée.

Cependant, 2 approches sont possible : l'exclusion ou l'inclusion

  • Exclusion on va ignorer une partie du code si les valeurs qu'il utilise n'ont pas bougées. (React...)
  • Inclusion le code va ignorer les parties du code par défaut, et l'on va ajouter uniquement le code que l'on souhaite voir recalculé (Svelte...)
